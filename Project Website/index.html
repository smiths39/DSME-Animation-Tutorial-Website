Data Structures Made Easy was designed to teach the intermediate programmer how to program and effectively use various data structures and algorithms. We invite you to begin the tutorials provided to assist you in your learning experience.

What the site offers

Information on essential data structures and algorithms

For those who wish to understand how a particular data structure or algorithm works, the information provided is split into categories. The content is designed to assist in maximising your understanding.

Animations

Interpret how a data structure or algorithm functions with the assistance of an interactive animation. Maximising your perception of how a data structure or algorithm functions will expand your ability to program it in any chosen programming language.

Downloadable documentation

For those who wish to download documentation based on a selected data structure or algorithm provided within the website. Documentation is provided in PDF format.

Downloadable Java programs

Included within the documentation is a program of the selected data structure or algorithm, which is incremented in the Java programming language. Documentation is provided in PDF format.

Screencasts

Video tutorials are provided to give an overview of all the information required to understand how a particular data structure or algorithm functions and how to program it.

Feedback

Our main priority is to make the information we provide as good as it can be. If you ever think a topic is not explained clearly or if you believe we should add a specific data structure or algorithm to our site, please let us know via our contact details and we'll try our best to improve on a specific topic.
